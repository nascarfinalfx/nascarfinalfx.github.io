Instalación de Python

  1. Si tiene el Archivo.py, descargue el instalador de Python. Es posible que pueda pedir permisos para Windows.
    • Instala Python 3.14 o superior: https://www.python.org/
    • Desde la carpeta del proyecto ejecuta: python -m pip install -r requirements.txt
    • Inicia: python main.py

Requisitos del sistema

A continuación se indican requisitos orientativos para el ejecutable Python/Pygame que acompaña al proyecto.

Mínimos
  • OS: Windows 10 / Ubuntu 20.04 o equivalente.
  • CPU: 2 núcleos modernos.
  • RAM: 2 GB.
  • Espacio: 50 MB libres.
  • Python: no requerido si usas .exe; si ejecutas desde fuente: Python 3.10+ y Pygame 2.x.
Recomendado
  • OS: Windows 10/11 o una distro Linux reciente.
  • CPU: 4 núcleos.
  • RAM: 4 GB+
  • GPU: aceleración 2D (drivers actualizados).
  • Python 3.10+ y Pygame 2.1+ si ejecutas desde fuente.
Detalles técnicos — dependencias (click)
▶
  • requirements.txt típico: pygame>=2.0.0, numpy (opcional)
  • Si empaquetas con PyInstaller: comprueba la inclusión de carpetas assets/ y pygame DLLs en Windows.
