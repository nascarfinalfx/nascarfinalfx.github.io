Instalación (ejecutable y código)

  1. Si tiene el archivo .py: Descargue Python Version 3.11.1
    • Instala Python 3.11.1 en:
      • Este lugar
    • Desde la carpeta de descarga ejecute instalador de Python
    • Una vez instalado. Vaya al logo de Windows y busque Windows Powershell, ponga pip install pygame
    • Una vez ya terminado, vaya a la carpeta de descargas o la que se encuentre nuestro juego y dele doble click para iniciar.
    • Si presenta dudas o problemas, comunicarse al correo nashecounter6@gmail.com

Requisitos del sistema

A continuación se indican requisitos orientativos para el ejecutable Python/Pygame que acompaña al proyecto.

Mínimos
  • OS: Windows 10 / Ubuntu 20.04 o equivalente.
  • CPU: 2 núcleos modernos.
  • RAM: 2 GB.
  • Espacio: 50 MB libres.
  • Python: no requerido si usas .exe; si ejecutas desde fuente: Python 3.10+ y Pygame 2.x.
Recomendado
  • OS: Windows 10/11 o una distro Linux reciente.
  • CPU: 4 núcleos.
  • RAM: 4 GB+
  • GPU: aceleración 2D (drivers actualizados).
  • Python 3.10+ y Pygame 2.1+ si ejecutas desde fuente.
